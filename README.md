# MVHS ClubConnect

## Description
MVHS ClubConnect is a **black-and-gold themed web application** designed to improve student life at Monta Vista High School. It centralizes all school activities into a **dynamic, interactive dashboard** where students can manage clubs, track events, organize homework, and quickly access important contacts. The platform also features **login/signup**, responsive design, search and sorting functions, and an AI-powered homework helper to assist students efficiently.

---

## Features

### Login & Signup
- Secure account creation with **form validation**.
- Personalized dashboard for each student.
- User data stored in **local storage**.

### Dashboard Sections
- **Home:** Welcome page with instructions and quick stats.
- **Clubs:** Add, edit, delete, search, and sort clubs by name, category, or upcoming meeting time. Clubs are displayed in **responsive cards** (1â€“2 per row depending on screen size).
- **Events:** Manage school events with sorting by upcoming dates. Events show **time, location, and organizers**.
- **Homework:** Track homework with due dates. Includes a **floating AI chatbot** for help and solutions.
- **Contacts:** Quick access to important school contacts like the school office or departments.
- **Analytics:** Visual summary of total clubs, joined clubs, favorite clubs, upcoming events, and pending homework.

### UI/UX
- **Black & gold theme** for modern, visually appealing interface.
- Hover effects, card shadows, smooth transitions, and animations.
- Fully **responsive design** with hamburger menu for mobile devices.
- Consistent **search bars and select filters** across all sections.

### AI Homework Helper
- Floating chatbot icon in the Homework section.
- Provides hints and guidance for solving homework problems.
- Can be connected to **GPT API or GitHub Copilot** for real AI responses.

## How to Use

### 1. Access the Application
- Open `index.html` in your browser (or deploy to a web host like GitHub Pages).

### 2. Create an Account
1. Click **Sign Up**.
2. Fill in your **first name, last name, email, and password**.
3. Click **Sign Up** to create an account.

### 3. Login
- Enter your registered **email and password** to access your personalized dashboard.

### 4. Navigate Dashboard
- Use the **navbar** to switch between **Home, Clubs, Events, Homework, Contacts, and Analytics**.
- The content changes dynamically without page reload.

### 5. Manage Clubs
- Add a new club using the **Club form**.
- Edit or delete clubs using the **buttons on each club card**.
- Search clubs by name and sort by category or upcoming meeting times.

### 6. Manage Events
- Add new events with **event name, date, time, location, and organizers**.
- Sort events by upcoming date.
- Edit or delete events directly from the event cards.

### 7. Manage Homework
- Add homework tasks with **subject and due date**.
- Use the **floating AI chatbot** to get help solving problems.

### 8. View Contacts
- Access school and department contacts easily.
- Includes **phone numbers and email addresses**.

### 9. Analytics
- See an overview of your **clubs, events, homework**, and activity stats.
- Helps you track participation and upcoming deadlines.
